# Portafolio Web - Jairo Jaramillo

Este es mi portafolio web

## Descripción

En este proyecto creé una página web donde muestro información sobre mí, mis habilidades, algunos trabajos que he realizado y un formulario de contacto.

El objetivo fue aplicar lo aprendido en clases de "Front End" y entender cómo estructurar una página web completa.

## Tecnologías usadas

- HTML
- CSS
- Git y GitHub

## Contenido de la página

- Sobre mí: información personal y presentación
- Habilidades: tecnologías que estoy aprendiendo
- Proyectos: trabajos realizados durante mi formación
- Contacto: formulario e información para comunicarse conmigo

## Proyectos incluidos

- Blog web hecho en clases con HTML y CSS
- Informe académico en Word
- Infografía sobre bases de datos SQL y NoSQL

## Características

- Diseño adaptable a distintos dispositivos (responsive)
- Uso de Flexbox y Grid
- Enlaces a redes sociales
- Estructura clara y organizada
